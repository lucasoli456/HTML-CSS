Pricipais formatações

Negrito / Destaque

Nessa frase temos um TERMO EM NEGRITO usando a tag B (Não semântica)

Nessa frase temos um TERMO EM NEGRITO usando a tag STRONG (Semântica)

Itálico / Ênfase

Nessa frase temos um TERMO EM ITÁLICO usando a tag I (Não semântica)

Nessa frase temos um TERMO EM ITÁLICO usando a tag EM (Semântica)

Texto Marcado

Podemos criar também um TEXTO MARCADO usando a tag MARK. (Semântica)

E no outro parágrafo, temos outro TEXTO MARCADO. (Semântica)

Texto grande e texto pequeno

Nesse parágrafo irei criar um TEXTO GRANDE usando a tag BIG. (Não semântica)

E nesse parágrafo irei criar um TEXTO PEQUENO usando a tag SMALL. (Semântica)

Texto Deletado

Podemos marcar um TEXTO COMO EXCLUÍDO para indicar que ele deve ser lido, mas não considerado usando a tag DEL. (Semântica)

Texto inserido

Podemos marcar um TEXTO COMO INSERIDO para das uma ênfase e indicar que ele foi adicionado depois usando a tag INS. (Semântica)

Existe também o sublinhado com a tag U. (Não semântica)

Texto sobrescrito

Para inserir coisas do tipo x20+3 usando a tag SUP. (Semântica)

Texto subscrito

Para inserir coisas do tipo H2O usando a tag SUB. (Semântica)
